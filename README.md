# PrivacyTestLab Website

Public source code for PrivacyTestLab.com, including privacy testing tools, educational resources, and browser security utilities.

## About PrivacyTestLab

PrivacyTestLab is an independent privacy and cybersecurity platform focused on helping users understand and improve their online privacy through browser-based testing tools, educational guides, and transparency-focused research.

Website: https://privacytestlab.com

## Features

* IP Address Detection
* DNS Leak Testing
* WebRTC Leak Detection
* Browser Fingerprint Analysis
* VPN / Proxy Detection
* Internet Speed Testing
* Ping Testing
* Metadata Removal Utilities
* Privacy Education Articles
* Browser Security Resources

## Project Goals

PrivacyTestLab aims to:

* Improve privacy awareness
* Promote transparency in online tracking
* Provide accessible privacy testing tools
* Help users identify privacy and security risks
* Support privacy-focused research and education

## Repository Scope

This repository contains the public-facing source code used by PrivacyTestLab.

Included:

* Frontend code
* Public website components
* Privacy testing tools
* Educational content
* Static assets
* Documentation

Not Included:

* Server infrastructure
* Administrative systems
* Databases
* Analytics configuration
* API keys and secrets
* Internal security controls
* Hosting and deployment configuration
